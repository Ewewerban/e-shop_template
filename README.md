# Simple Attraction & Service Catalog Template

A clean, static HTML/CSS template designed for free hosting on **GitHub Pages**.

## How to Start in 3 Steps:
1. Click the **Fork** button in the upper right corner of this repository to copy it to your account.
2. Go to your forked repository's **Settings** -> **Pages**, change the `Branch` source to `main`, and click **Save**. Your website will be live at the URL provided by GitHub.
3. Edit the files directly on GitHub or locally to update texts, prices, and swap image files inside the `assets/img/` directory.

## How to Add a New Product:
1. Create a new folder matching the category and item name, for example: `cat1/product-1/`.
2. Copy the content of `product-template.html` into a new file named `index.html` inside that new folder.
3. Update the description, technical specs table, and pricing inside that new file.
4. Open the main root `index.html` file and add a new product card pointing to your newly created folder path.
